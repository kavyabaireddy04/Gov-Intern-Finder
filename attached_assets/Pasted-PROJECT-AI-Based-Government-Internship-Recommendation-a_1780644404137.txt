PROJECT: AI-Based Government Internship Recommendation and Eligibility Analysis System (100% Free, No API Keys)

Modify and complete this project so it runs entirely on Replit Free Tier without using OpenAI, Gemini, Claude, Anthropic, or any paid AI service.

GOAL
Build a web application that recommends suitable Government Internships to students based on:

- GPA
- Qualification
- Skills
- Preferred Location

The system must analyze eligibility, calculate match scores, rank internships, and display the best recommendations.

DATA SOURCE

The user will upload internship CSV files extracted from:

- AICTE Internships
- PM Internship Portal
- Government Internship Portals
- Other government internship sources

One CSV named:

india_gov_internship_sources.csv

contains verified government internship sources.

TASK 1: DATA CLEANING AND DATASET CREATION

When internship CSV files are uploaded:

1. Read all uploaded CSV files.
2. Identify which internships belong to government organizations.
3. Use india_gov_internship_sources.csv as the primary verification source.
4. Remove duplicates.
5. Ignore invalid records.
6. Normalize column names.
7. Create one final dataset.

Required columns:

- internship_title
- organization
- skills_required
- min_gpa
- qualification
- location
- deadline
- internship_url
- source
- description

Rules:

- Every row must contain all required fields.
- Missing values should be handled intelligently.
- Remove duplicate internships.
- Keep only active internships whenever possible.

Save as:

final_government_internships.csv

TASK 2: REAL-TIME GOVERNMENT INTERNSHIP SCRAPING

Build rule-based scrapers using:

- Requests
- BeautifulSoup4
- Pandas
- PDFPlumber or PyPDF2

Supported sources:

- AICTE Internship Portal
- PM Internship Portal
- NITI Aayog Internship Portal
- Government Ministry Internship Pages
- Other websites listed in india_gov_internship_sources.csv

Requirements:

- No AI APIs
- No LLMs
- No API Keys

Features:

- Crawl websites directly
- Extract internship details from HTML pages
- Extract internship details from PDF notices
- Follow pagination automatically
- Remove duplicates
- Update database automatically
- Generate CSV export automatically
- Handle website failures gracefully

TASK 4: ELIGIBILITY ANALYSIS

Student Inputs:

- GPA
- Qualification
- Skills
- Preferred Location

Eligibility Rules:

- GPA check
- Qualification check
- Location preference check
- Required skill check

Output:

- Eligible / Not Eligible
- Eligibility Percentage

TASK 5: SKILL MATCHING

Use:

- TF-IDF Vectorizer
- Cosine Similarity

Process:

1. Convert student skills into text.
2. Convert internship skills into text.
3. Calculate similarity score.
4. Generate Skill Match Percentage.

TASK 6: MATCH SCORING

Create weighted scoring:

40% Skill Match
25% GPA Match
20% Qualification Match
15% Location Match

Formula:

Final Score =
(0.40 × Skill Score) +
(0.25 × GPA Score) +
(0.20 × Qualification Score) +
(0.15 × Location Score)

Return score from 0–100.

TASK 7: RECOMMENDATION ENGINE

For each student:

1. Check eligibility.
2. Calculate TF-IDF similarity.
3. Calculate weighted score.
4. Rank internships.
5. Return Top 10 internships.

Display:

- Internship Title
- Organization
- Match Percentage
- Eligibility Status
- Deadline
- Location
- Apply Link

Sort by highest score first.
